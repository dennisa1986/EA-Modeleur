# Prompt Library

Reusable prompts for AI-assisted pipeline steps (ingestion, extraction, validation review).

Naming: `<stage>-<purpose>.md`

Examples:
- `ingest-extract-elements.md` — prompt for extracting model elements from text
- `validation-explain-finding.md` — prompt for explaining a ValidationFinding
- `serialization-review-xmi.md` — prompt for reviewing generated XMI
